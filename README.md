# KeenKeeper - Personal Friendship Tracker

**KeenKeeper** is a modern personal CRM designed to help you nurture and maintain your most important relationships. It tracks your interactions and reminds you when it's time to reach out to a friend.

## 🚀 Key Features
- **Real-time Interaction Tracking:** Log calls, texts, and video chats directly from a friend's profile and see them updated instantly in your timeline.
- **Friendship Analytics:** Visualize your communication patterns with a dynamic Pie Chart showing the distribution of your interaction types.
- **Smart Status System:** Automatically categorizes friends into "On-Track", "Almost Due", or "Overdue" based on your last contact date.

## 🛠️ Technologies Used
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **State Management:** React Context API
- **Charts:** Recharts
- **Notifications:** React Hot Toast
- **Icons:** SVG & Lucide Icons

## 📱 Responsive Design
The application is fully responsive and optimized for mobile, tablet, and desktop screens.
